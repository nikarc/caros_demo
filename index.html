CarOS

What is it?

CarOS is a front end application designed to control a carputer. CarOS is targeted for Raspbian and the Raspberry Pi, however, it is built using Electron which allows some degree of portability between all the major OS's.

Try it out!

How It Works

  • On first startup, it scans a chosen directory for mp3’s and grabs the id3 data with an id3-parser.

  • If there’s album art, it creates a jpg image and saves.

  • Then it saves all the song information to a local database.

  • That's it! Start enjoying your music in your car!
